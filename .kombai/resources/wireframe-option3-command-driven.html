Option 3 — Command-Driven: Command Bar persistente no topo (busca ações + módulos + registros). Sidebar slim icon-rail (44px) como backup visual. Dashboard agenda-style: pendências + resumo + feed ao vivo + atalhos rápidos com teclas. Módulos usam expand-in-place (sem modais): clicar na linha expande detalhes E formulário de edição inline. Density compacta (tabelas densas, minimal chrome). Keyboard-centric: F2-F12 shortcuts, ↑↓ navegar, Enter expandir. Novos componentes: PersistentCommandBar, QuickActionsStrip, AgendaDashboard, LiveFeed, InlineExpandRow, ContextualSubNav, ShortcutsPanel. Reutiliza: KpiCard, Tabs, StatusBadge, Pagination, CommandPalette (enhanced).
